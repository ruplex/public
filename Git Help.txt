Стандартные команды:
git clone "ссылка на репозиторий" - клонирует репозиторий;

git remote add "имя" "путь к репозиторию" - добавить репозиторий;

git init - иницилизирует файлы конфиги;

git add "имя файла";
git add . ;
git add --all;

git commit -m 'коммит'

git push (-f) "имя репозитория" master;

git pull - скачивает измененные файлы

git rm - удаляет файлы ;

Дополнительно:
git config core.autocrlf false - решает проблемы с CRLF;

git stash - обновляет коммиты;
git reset --hard HEAD - обновляет коммиты;

git help - помощь;

Если команда git pull не работает:
git add --all
git commit -m ''
git pull
